Гра являється копією на свій лад відомого Bomberman 2 для нінтендо. 
Для того, щоб пограти, потрібно розархівувати Release.7z і запустити .exe файл.
Для того, щоб працювати з кодом, потрібно підключити бібліотеку SFML, що знаходиться 
в папці libery. Детальніше про бібліотеку https://www.sfml-dev.org/
